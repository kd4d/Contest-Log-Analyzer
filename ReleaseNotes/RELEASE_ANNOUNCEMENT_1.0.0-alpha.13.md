# Release Announcement: v1.0.0-alpha.13

**Release Date:** January 26, 2026  
**Version:** `v1.0.0-alpha.13`

---

## Highlights

This release focuses on **UI improvements and report layout fixes** to make the interface more user-friendly and reports easier to read.

---

## Improvements

### Enhanced UI Visibility
**Improved tab labels and checkbox visibility**

- Tab labels on the home page are now much larger and bold for better visibility
- "Public Log Archive" tab now clearly labeled as "Upload from Public Log Archive"
- Checkbox borders are now darker and more visible

### Better Report Display
**Fixed layout and rendering issues**

- Removed unnecessary scrollbars from report cards
- Improved report card layout and resizing
- Reports now use full width for better readability
- Fixed alignment issues in QSO breakdown reports

### Performance Improvements
**Faster report loading**

- Replaced iframes with direct text rendering in Multiplier Dashboard Analysis Reports
- Reports now load faster and display more reliably

---

## New Features

- Added InactiveTime column to breakdown reports for better timing analysis
- Improved zone normalization and CQ 160 mode display
