# Release Announcement: v1.0.0-alpha.14

**Release Date:** January 26, 2026  
**Version:** `v1.0.0-alpha.14`

---

## Highlights

This release fixes **critical dashboard issues for single-log analysis** that were preventing users from viewing important reports.

---

## Important Fixes

### Rate Detail Tab Now Works
**Fixed 404 error on Rate Detail tab for single logs**

- The Rate Detail tab on the QSO Dashboard now works correctly for single-log analyses
- Previously showed a "Report not found" error
- Now displays rate sheet data correctly

### Multiplier Breakdown Reports Fixed
**Fixed blank reports for single logs**

- Multiplier breakdown reports (Countries/Zones) now display correctly for single-log analyses
- Previously showed blank boxes in the Multiplier Dashboard
- Now shows complete breakdown data as expected
