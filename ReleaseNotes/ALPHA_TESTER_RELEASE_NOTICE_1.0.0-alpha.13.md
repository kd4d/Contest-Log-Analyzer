# Alpha Release Notice: v1.0.0-alpha.13

**Release Date:** January 26, 2026  
**Version:** v1.0.0-alpha.13

---

## What's New

This release focuses on making the interface easier to use and fixing some display issues you may have noticed.

### Easier to Navigate

**Bigger, clearer tab labels**
- The "Manual Log Upload" and "Upload from Public Log Archive" tabs are now much larger and bold
- You'll find it easier to see which section you're in at a glance

**Better checkbox visibility**
- The "Apply WRTC Scoring Rules" checkbox now has a darker border, making it easier to see

### Report Display Improvements

**Cleaner report layout**
- Removed unnecessary scrollbars from report cards
- Reports now use the full width of the screen for better readability
- Fixed alignment issues in QSO breakdown reports (the '/' symbols now line up properly)

**Faster report loading**
- Replaced iframes with direct text rendering in Multiplier Dashboard Analysis Reports
- Reports should load faster and display more reliably

### New Features

- Added InactiveTime column to breakdown reports for better timing analysis
- Improved zone normalization and CQ 160 mode display

---

## What This Means for You

**If you're using the web interface:**
- The home page will be easier to navigate with the larger tab labels
- Reports should look cleaner and be easier to read
- You may notice faster loading times for some reports

**No action required** - All improvements are automatically applied. Just refresh your browser if you have the site open.
