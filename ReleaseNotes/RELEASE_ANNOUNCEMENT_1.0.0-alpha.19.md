# Release Announcement: v1.0.0-alpha.19

**Release Date:** June 18, 2026  
**Version:** `v1.0.0-alpha.19`

---

## Highlights

This release improves **missed multiplier text reports** for multi-mode contests: cells now show **how** and **on which mode** a competitor worked each multiplier (e.g. `(Run/CW) 2`, `(S&P/PH) 1`) with **aligned columns**. Per-mode reports (ARRL 10) keep the existing `(Run)` / `(S&P)` format. Also included: **ARRL Sweepstakes** public log download, **Run/S&P algorithm** documentation updates, and **RBN download** tooling for IARU archives.

---

## What's New

### Missed Multipliers (Multi-Mode)
- Combined all-modes reports (IARU-HF, WRTC): aligned mode slots in each callsign column.
- Per-mode reports: no change to cell format; mode stays in the report header.

### Public Logs
- ARRL Sweepstakes CW and SSB logs can be fetched from the public archive.

### Documentation & Tooling
- Run/S&P classification algorithm spec updated.
- RBN download script and IARU 2025 daily zip archives for offline data workflows.
